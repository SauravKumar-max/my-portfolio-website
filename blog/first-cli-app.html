Build Your First CLI Application

A CLI (command line interface) Applications are programs that you interact with entirely through your console. They are totally opposite of GUI (graphical user interface) as they have no graphics beyond what you see in your terminal after you run the program.

Prerequisites

  • Basic knowledge of programming.
  • Basic knowledge of Javascript & Nodejs.
  • Before starting You can play this game to get the essence of a CLI app.

Installation and Setup

  • First you need an IDE (Integrated Development Environment) or a code editor.
    1. Visual Studio Code.
    2. Repl - An online IDE, use this if you are a beginner.
  • Download and Install Node.js.
